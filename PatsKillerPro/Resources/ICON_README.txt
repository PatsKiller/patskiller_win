PatsKiller Pro Icon Requirements
================================

Place your application icon here as: app.ico

Requirements:
- Format: Windows ICO
- Sizes: Include multiple sizes (16x16, 32x32, 48x48, 256x256)
- Color depth: 32-bit with alpha transparency

Recommended tools for creating icons:
- GIMP (free): https://www.gimp.org/
- IcoFX: https://icofx.ro/
- RealWorld Icon Editor: http://www.rw-designer.com/icon-editor
- Online converters: https://convertio.co/png-ico/

Creating from PNG:
1. Create a 256x256 PNG with transparent background
2. Use a converter to create ICO with multiple sizes
3. Save as app.ico in this folder

Placeholder:
If no icon is provided, the build will use the default Windows application icon.
